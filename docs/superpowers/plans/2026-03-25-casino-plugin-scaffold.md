> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the monolithic Maven prototype with a clean, interface-driven Gradle KTS plugin scaffold that four developers can extend independently.

**Architecture:** Constructor injection (Approach B) with a single composition root in `CasinoPlugin.onEnable()`. All components receive dependencies via constructor; no static singletons. New games are registered in `GameRegistrar.java` only — no other file touched.

**Tech Stack:** Java 21, Paper API 1.21.4, Gradle 8.10 KTS, Lombok, FoliaLib, bStats (shaded), JUnit 5 + MockBukkit 3.x.

package dev.casino.economy;

import org.bukkit.entity.Player;

/**
 * Service contract for the gold economy.
 * All amounts are expressed in gold ingot units
 * (1 block = 9 ingots, 9 nuggets = 1 ingot, floor division).
 */
public interface EconomyManager {

    /** Returns total gold in ingots: inventory + Enderchest combined. */
    int getBalance(Player player);

    /**
     * Withdraws {@code amount} ingots from the player.
     * Draws from inventory first, then Enderchest.
     * Consumes smallest denominations first (nuggets → ingots → blocks).
     *
     * @return {@code false} if the player has insufficient funds; no items removed.
     */
    boolean withdraw(Player player, int amount);

    /**
     * Deposits {@code amount} ingots into the player's inventory (as ingots).
     * Falls back to Enderchest if inventory is full; drops at feet if both full.
     */
    void deposit(Player player, int amount);

    /** Returns {@code true} if {@link #getBalance(Player)} >= {@code amount}. */
    boolean hasFunds(Player player, int amount);
}
```

/**
 * Contract for every casino game.
 * <p>
 * Implement this interface and register in {@code GameRegistrar} to add a new game.
 * {@link #stop(Player)} must be idempotent: calling it when no session is active is safe.
 */
public interface CasinoGame {

    /**
     * Starts a game session for the given player.
     * Implementations typically open a game-specific GUI.
     */
    void start(Player player);

    /**
     * Ends the game session for the given player.
     * Called by: PlayerQuitListener, CasinoPlugin.onDisable(), or the game itself.
     * Must be idempotent (safe to call with no active session).
     */
    void stop(Player player);

    /** Icon shown in the main casino menu. Must not return {@code null}. */
    ItemStack getIcon();

    /** Unique snake_case identifier used as the registry key (e.g. "slot_machine"). */
    String getName();

    /** Display name rendered on the icon in the main menu. */
    Component getDisplayName();
}

     * and opens the inventory for the player.

     * Called by PlayerQuitListener and CasinoPlugin.onDisable().

package dev.casino.games;

import java.util.Collection;
import java.util.Collections;
import java.util.LinkedHashMap;
import java.util.Map;
import java.util.Optional;

/** Map-based {@link GameRegistry} that preserves registration order. */
public final class DefaultGameRegistry implements GameRegistry {

    private final Map<String, CasinoGame> games = new LinkedHashMap<>();

    @Override
    public void register(CasinoGame game) {
        games.put(game.getName(), game);
    }

    @Override
    public Collection<CasinoGame> getAll() {
        return Collections.unmodifiableCollection(games.values());
    }

    @Override
    public Optional<CasinoGame> findByName(String name) {
        return Optional.ofNullable(games.get(name));
    }
}

public final class GoldEconomyManager implements EconomyManager {

    @Override
    public int getBalance(Player player) {
        return toIngots(player.getInventory()) + toIngots(player.getEnderChest());
    }

    @Override
    public boolean hasFunds(Player player, int amount) {
        return getBalance(player) >= amount;
    }

    @Override
    public boolean withdraw(Player player, int amount) {
        if (!hasFunds(player, amount)) return false;
        int remaining = removeGold(player.getInventory(), amount);
        if (remaining > 0) removeGold(player.getEnderChest(), remaining);
        return true;
    }

    @Override
    public void deposit(Player player, int amount) {
        ItemStack ingots = new ItemStack(Material.GOLD_INGOT, amount);
        Map<Integer, ItemStack> overflow = player.getInventory().addItem(ingots);
        if (overflow.isEmpty()) return;
        Map<Integer, ItemStack> enderOverflow = player.getEnderChest().addItem(overflow.get(0));
        if (!enderOverflow.isEmpty()) {
            player.getWorld().dropItemNaturally(player.getLocation(), enderOverflow.get(0));
        }
    }

    // ── Private helpers ────────────────────────────────────────────────────────

    /** Total ingot value of the inventory (floor division). */
    private int toIngots(Inventory inventory) {
        return sum(inventory, Material.GOLD_NUGGET) / 9
             + sum(inventory, Material.GOLD_INGOT)
             + sum(inventory, Material.GOLD_BLOCK) * 9;
    }

    /**
     * Removes {@code ingotAmount} ingots' worth from {@code inventory},
     * smallest denomination first (nuggets → ingots → blocks).
     *
     * @return ingots still unpaid (0 = fully removed); caller withdraws remainder from Enderchest.
     */
    private int removeGold(Inventory inventory, int ingotAmount) {
        int nuggetTarget = ingotAmount * 9; // work in nugget units

        // 1. Consume nuggets
        int aN = sum(inventory, Material.GOLD_NUGGET);
        int tN = Math.min(aN, nuggetTarget);
        consume(inventory, Material.GOLD_NUGGET, tN);
        nuggetTarget -= tN;
        if (nuggetTarget <= 0) return 0;

        // 2. Consume ingots (ceiling-divide so we cover remaining nugget target)
        int aI = sum(inventory, Material.GOLD_INGOT);
        int tI = Math.min(aI, (nuggetTarget + 8) / 9);
        consume(inventory, Material.GOLD_INGOT, tI);
        nuggetTarget -= tI * 9;
        if (nuggetTarget <= 0) { returnChange(inventory, -nuggetTarget); return 0; }

        // 3. Consume blocks
        int aB = sum(inventory, Material.GOLD_BLOCK);
        int tB = Math.min(aB, (nuggetTarget + 80) / 81);
        consume(inventory, Material.GOLD_BLOCK, tB);
        nuggetTarget -= tB * 81;
        if (nuggetTarget <= 0) { returnChange(inventory, -nuggetTarget); return 0; }

        return (nuggetTarget + 8) / 9; // remaining ingots the caller must take elsewhere
    }

    /** Gives back excess nugget units as ingots + nuggets. */
    private void returnChange(Inventory inventory, int excessNuggets) {
        if (excessNuggets <= 0) return;
        int ingots  = excessNuggets / 9;
        int nuggets = excessNuggets % 9;
        if (ingots  > 0) inventory.addItem(new ItemStack(Material.GOLD_INGOT,  ingots));
        if (nuggets > 0) inventory.addItem(new ItemStack(Material.GOLD_NUGGET, nuggets));
    }

    private int sum(Inventory inventory, Material material) {
        return inventory.all(material).values().stream()
                .mapToInt(ItemStack::getAmount).sum();
    }

    private void consume(Inventory inventory, Material material, int amount) {
        if (amount <= 0) return;
        ItemStack[] contents = inventory.getContents();
        for (int i = 0; i < contents.length && amount > 0; i++) {
            ItemStack item = contents[i];
            if (item == null || item.getType() != material) continue;
            int take = Math.min(item.getAmount(), amount);
            item.setAmount(item.getAmount() - take);
            if (item.getAmount() == 0) inventory.setItem(i, null);
            amount -= take;
        }
    }
}

import net.kyori.adventure.text.Component;
import org.bukkit.Bukkit;
import org.bukkit.event.inventory.InventoryClickEvent;
import org.bukkit.inventory.Inventory;

import java.util.HashMap;
import java.util.Map;

/**
 * Fluent builder for casino GUI inventories.
 * <p>
 * Usage:
 * <pre>
 *   GuiBuilder builder = GuiBuilder.create(Component.text("Casino"), 27)
 *       .item(11, GuiItem.of(icon, event -> game.start(player)))
 *       .item(13, GuiItem.decoration(Material.GRAY_STAINED_GLASS_PANE));
 *   Inventory inv = builder.build();
 * </pre>
 * Retain the builder instance to route clicks via {@link #handleClick(InventoryClickEvent)}.
 */
public final class GuiBuilder {

    private final Component title;
    private final int size;
    private final Map<Integer, GuiItem> slots = new HashMap<>();

    private GuiBuilder(Component title, int size) {
        if (size < 9 || size > 54 || size % 9 != 0) {
            throw new IllegalArgumentException("GUI size must be a multiple of 9 between 9 and 54, got: " + size);
        }
        this.title = title;
        this.size  = size;
    }

    public static GuiBuilder create(Component title, int size) {
        return new GuiBuilder(title, size);
    }

    /** Places a {@link GuiItem} at the given slot index (0-based). */
    public GuiBuilder item(int slot, GuiItem item) {
        slots.put(slot, item);
        return this;
    }

    /** Builds the Bukkit {@link Inventory} with all configured items. */
    public Inventory build() {
        Inventory inventory = Bukkit.createInventory(null, size, title);
        slots.forEach((slot, guiItem) -> inventory.setItem(slot, guiItem.getItemStack()));
        return inventory;
    }

    /**
     * Routes a click event to the {@link GuiItem} at the clicked slot.
     * No-op if the slot has no configured item.
     */
    public void handleClick(InventoryClickEvent event) {
        GuiItem item = slots.get(event.getRawSlot());
        if (item != null) item.executeAction(event);
    }
}

package dev.casino.gui;

import org.bukkit.entity.Player;
import org.bukkit.event.inventory.InventoryClickEvent;

import java.util.Collections;
import java.util.Set;
import java.util.UUID;
import java.util.concurrent.ConcurrentHashMap;

/** Thread-safe (Folia-ready) implementation of {@link GuiManager}. */
public final class DefaultGuiManager implements GuiManager {

    private final ConcurrentHashMap<UUID, GuiMenu> activeMenus = new ConcurrentHashMap<>();

    @Override
    public void open(Player player, GuiMenu menu) {
        activeMenus.put(player.getUniqueId(), menu);
        player.openInventory(menu.buildInventory(player));
    }

    @Override
    public void handleClick(InventoryClickEvent event) {
        if (!(event.getWhoClicked() instanceof Player player)) return;
        GuiMenu menu = activeMenus.get(player.getUniqueId());
        if (menu == null) return; // not a casino inventory — do NOT cancel
        event.setCancelled(true);
        menu.handleClick(event);
    }

    @Override
    public void cleanup(UUID playerId) {
        activeMenus.remove(playerId);
    }

    @Override
    public Set<UUID> getActivePlayers() {
        return Collections.unmodifiableSet(activeMenus.keySet());
    }
}

# Contributing — How to Add a New Casino Game

This guide explains how to add a new game so it appears in the main casino menu
without touching any shared plugin code.

---

## Prerequisites

- Java 21+
- Run `./gradlew build` successfully before starting.
- Read the `CasinoGame` interface at `src/main/java/dev/casino/games/CasinoGame.java`.

---

## Step 1: Create your game class

Create a new package for your game under `dev.casino.games.<yourgame>`:

```
src/main/java/dev/casino/games/slots/SlotMachine.java
```

Implement the `CasinoGame` interface:

```java
package dev.casino.games.slots;

import dev.casino.core.PluginContext;
import dev.casino.games.CasinoGame;
import net.kyori.adventure.text.Component;
import org.bukkit.Material;
import org.bukkit.entity.Player;
import org.bukkit.inventory.ItemStack;
import org.bukkit.inventory.meta.ItemMeta;

public final class SlotMachine implements CasinoGame {

    private final PluginContext context;

    public SlotMachine(PluginContext context) {
        this.context = context;
    }

    @Override
    public void start(Player player) {
        // Open your game GUI here using context.gui() and GuiBuilder
    }

    @Override
    public void stop(Player player) {
        // Clean up session state. Must be safe to call with no active session.
    }

    @Override
    public ItemStack getIcon() {
        ItemStack item = new ItemStack(Material.LEVER);
        ItemMeta meta  = item.getItemMeta();
        meta.displayName(getDisplayName());
        item.setItemMeta(meta);
        return item;
    }

    @Override
    public String getName() {
        return "slot_machine"; // unique snake_case key
    }

    @Override
    public Component getDisplayName() {
        return Component.text("Slot Machine");
    }
}
```

## Design rules

| Rule | Why |
|------|-----|
| Never call `Bukkit.createInventory()` directly | Use `GuiBuilder` instead |
| Never open an inventory with `player.openInventory()` directly | Use `GuiManager.open()` |
| `stop()` must be idempotent | Called on disconnect and server shutdown |
| All gold transactions through `EconomyManager` | Keeps economy logic in one place |
| Keep your game in its own package | Prevents accidental coupling |
